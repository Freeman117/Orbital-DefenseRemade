UtH Engine v0.10 (Pre-release) - See attached Mit license

Android building will be added to Releases TemplateProject in v1.0
Until then Android version can be builded with Master branch of UtH Engine.

NOTE: Run CopyAssets.bat to copy assets to required directories from ROOT/assets/

Changes from 0.9:
* Changed Math-library
* Added Gyro & Accelometer support for Android input
* New texture-file support: PNG 8-bit, JPEG baseline (non-progressive)
* Updated engine directory filepaths
* Changes in the interfaces of  Resource manager, Sprite batch & Particle system
